# Roll And Rate 🎲

**Live Project:** [https://collection-manager-website.onrender.com](https://collection-manager-website.onrender.com)

A dynamic board game collection manager that turns subjective play experiences into objective, weighted rankings.

---

## 📌 Project Overview
**Roll And Rate** is a web-based application designed for board game enthusiasts who want to move beyond simple five-star ratings. By utilizing a custom weighted-scoring algorithm, the platform helps users quantify their gaming preferences, manage their collections, and store personal notes—all within a clean, responsive interface.

![Project Preview](/docs/Project_Preview.bnp)

---

## 🛠️ Technical Toolkit & Logic Applied

### Backend Engine
* Flask web framework with server-side session management.

### Database Architecture
* SQL-based storage for persistent user collections and notes.

### Weighted Ranking Algorithm
* A custom-engineered formula that normalizes user inputs—heavily prioritizing *Fun* and *Strategy/Theme/Ease*—to produce a refined leaderboard.

---

## 📋 Core Functionality Matrix

### 1. Intelligent Weighted Ranking
Automatically calculates and sorts games based on the custom "essence" formula.

### 2. Dynamic Collection Dashboard
A sortable, scannable overview of your gaming library with individual rating breakdowns.

### 3. Session-Locked Privacy
Secure registration and persistent, encrypted user sessions.
